# Lecture 6 task

###Prerequisites:
__Windows users only:__
Download and install [Git Bash](https://git-scm.com/download/win)


__Install nvm:__
  + [Windows](https://github.com/coreybutler/nvm-windows)
  + [Unix](https://github.com/creationix/nvm)
 
__Install nodejs:__
```bash
nvm install 8.11.1
```

__Install yarn:__
```bash
npm install -g yarn
```

###Project setup:
__Open terminal:__

```bash
cd /path/to/lecture_06
```

```bash
yarn install
```
